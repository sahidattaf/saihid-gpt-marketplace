## How to Use These Files

1. Open [ChatGPT → Explore GPTs → Create](https://chat.openai.com/gpts/editor)
2. Paste the content of the **GPT Builder Instructions** section into the "Instructions" field
3. Set the name, description, and conversation starters from this file
4. Upload any listed **Knowledge Files**
5. Enable the recommended **capabilities** (browsing, code interpreter, image generation)
6. Publish and copy the link back into `lib/gptData.ts`

## Updating a GPT Link

When a GPT is published in ChatGPT:

1. Copy the GPT link from ChatGPT
2. Update the **GPT Builder Link** section in this file
3. Update the corresponding `link` field in `lib/gptData.ts`
4. Run `npm run build` and push
